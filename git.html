Comandos Básicos do Git

Quais são os principais comandos do Git que são usados com frequência?

  • git init: Inicializa um novo repositório Git em um diretório existente.
  • git clone: Cria uma cópia local de um repositório Git remoto.
  • git add: Adiciona mudanças no diretório de trabalho ao índice (staging area).
  • git commit: Registra as mudanças feitas no repositório com uma mensagem descritiva.
  • git push: Envia commits locais para o repositório remoto.
  • git pull: Busca e integra mudanças do repositório remoto para o repositório local.

Como fazer commits e qual a importância de uma mensagem de commit clara e descritiva?

Fazer commits no Git envolve registrar as alterações feitas nos arquivos do seu projeto. Mensagens de commit claras e descritivas são fundamentais para o controle de versão e colaboração eficaz.

Clonando Repositórios

O que significa clonar um repositório e como isso é feito?

Significa criar uma cópia local exata de um repositório remoto no seu computador. Para clonar um repositório Git, obtenha o URL de clonagem na página principal do repositório no GitHub e use o comando: 

git clone URL_do_repositório

Qual é a utilidade de clonar um repositório?

Isso permite que você trabalhe com o código localmente, faça alterações, desenvolva novos recursos e contribua para o projeto.
